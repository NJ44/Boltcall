## Purpose

Give personalized, data-grounded LinkedIn strategy — not generic advice. Every recommendation must cite specific numbers from the growth playbook (e.g., "30+ engagements in 45 min", "19% profile conversion target") and be tailored to the user's niche and current situation.

## Step-by-Step Workflow

### Step 1 — Load reference files
Read ALL of these before responding:
- `references/growth-playbook.md` — Matt Gray 468K followers system, core tactics
- `references/ai-workflow.md` — Lara Acosta 30-min/week AI content workflow
- `references/platform-shift-2026.md` — Current platform dynamics and what's changing
- `references/algorithm-mechanics.md` — Deep algorithm rules, signals, timing

If the Obsidian MCP is available, also pull these live vault notes:
- `LinkedIn MOC` (from the LinkedIn vault folder) — the master index
- Any other note linked from the MOC relevant to the user's question

### Step 2 — Load brand context
Read `skills/linkedin-brand-context/resources/brand-identity-template.md` and `skills/linkedin-brand-context/resources/company-template.md` to tailor all strategy to the user's specific niche, ICP, stage of growth, and offer.

### Step 3 — Assess the question
Identify the focus area and any data the user has shared (follower count, engagement rates, recent posts, etc.). If no data is shared, ask one clarifying question before giving strategy.

### Step 4 — Deliver strategy
Structure the response:
1. **Current state assessment** (if data provided)
2. **Core strategy** — specific, numbered steps with exact targets
3. **Priority order** — what to do first, second, third this week
4. **What NOT to do** — common mistakes from the playbook
5. **Quick win** — one thing they can do in under 10 minutes

Always cite the strategy source for each recommendation:
- 📘 Growth Playbook (Matt Gray)
- 🎯 AI Workflow (Lara Acosta)
- 📊 Algorithm Mechanics
- 🔄 Platform Shift 2026

## Key Numbers (always reference these)
- Profile-to-follower conversion target: 19%+
- Early engagement window: first 45 minutes
- Minimum engagement for algorithmic push: 30+ reactions/comments
- Link placement: comments or 45-min post edit — NEVER in post body
- Follower growth benchmark: 11,000+/month at scale
- Best posting days: Tue, Wed, Thu
- Best posting times: 8-10am or 12-1pm

## Constraints

- Never give strategy without reading the reference files first
- Always cite which source the advice comes from
- Never recommend scheduling tools — LinkedIn penalizes third-party schedulers
- Always include exact numbers, not vague guidance
- If the user shares analytics, give specific diagnosis before general advice
