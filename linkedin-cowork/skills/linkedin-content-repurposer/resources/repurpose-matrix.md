# Repurpose Matrix — Content Type → LinkedIn Format Mapping

> Not all content repurposes the same way. The source format determines which insight extraction method works best, and which LinkedIn format will perform. Use this matrix to match source → method → output.

---

## The Core Repurposing Principle

Repurposing is NOT:
- Copying the source and reformatting it
- Summarizing the content
- Pulling every point and turning it into a list

Repurposing IS:
- Extracting the **single sharpest insight** from the source
- Rewriting it from your perspective, in your voice
- Making it feel like it came from personal experience, not from a piece of content

The source is raw material. The output should feel like an original post.

---

## The Repurpose Matrix

### 1. Blog Post → LinkedIn

| | |
|---|---|
| **Extraction method** | Find the single most contrarian claim or the core framework. Ignore the rest. |
| **Best LinkedIn format** | Hook Stack (if single insight) or Long-Form List (if framework with 5+ steps) |
| **Hook angle** | Lead with the counter-intuitive claim or the result the framework produces |
| **What to cut** | Introduction, background, conclusion — everything that "sets up" the insight |
| **What to amplify** | The one thing the blog post builds to — make that the opening line |

**Example:**
Blog post about "How I scaled my email list from 0 to 10K" → extract: the single most surprising tactic → open with that tactic, not with the story of the list growth

---

### 2. YouTube Transcript → LinkedIn

| | |
|---|---|
| **Extraction method** | Find the story beat (the moment something changed) or the system reveal (the exact how) |
| **Best LinkedIn format** | Personal Story (if story beat) or Deep-Work System Reveal (if system/process) |
| **Hook angle** | Start in the moment — not "In this video I talked about..." |
| **What to cut** | Intro/outro, filler ("uh", "so basically"), anything that only makes sense in video context |
| **What to amplify** | The single insight that would make a viewer stop mid-video and say "wait, say that again" |

**Example:**
YouTube transcript from a 45-minute productivity video → extract: the one counterintuitive thing said at minute 22 → write a 250-word story post about that insight, from your perspective

---

### 3. Tweet Thread → LinkedIn

| | |
|---|---|
| **Extraction method** | Compress the entire thread to its 3 best points — discard the rest |
| **Best LinkedIn format** | Hook Stack (if the points are standalone) or Long-Form List (if they build on each other) |
| **Hook angle** | Take the best tweet from the thread and rewrite it as a LinkedIn hook |
| **What to cut** | Thread numbering, @mentions, Twitter-specific language ("RT if you agree") |
| **What to amplify** | The most shareable, most surprising, most specific insight from the thread |

**Example:**
15-tweet thread on "How I 10x'd my pricing" → extract: the 3 most surprising insights → write a Hook Stack with those 3 as the core statements, add your own context to each

---

### 4. Newsletter Issue → LinkedIn

| | |
|---|---|
| **Extraction method** | Extract the core framework, tip, or lesson. Ignore the preamble and closing. |
| **Best LinkedIn format** | Carousel (if framework has 5+ discrete components) or Long-Form List |
| **Hook angle** | Lead with the result the framework produces — not the framework itself |
| **What to cut** | Newsletter context ("this week I wanted to share"), links to external content, self-promotion |
| **What to amplify** | The framework name (if it has one) + the outcome it creates |

**Example:**
Newsletter issue about a 5-part content planning system → extract: the 5 steps → build a 7-slide carousel (Title + 5 value slides + CTA)

---

### 5. Podcast Clip / Transcript → LinkedIn

| | |
|---|---|
| **Extraction method** | Pull the most quotable insight or the specific "aha moment" the host/guest had |
| **Best LinkedIn format** | Data+identity hook → Long-Form Post (if single insight) |
| **Hook angle** | Open with the insight reframed as a claim — not "On a recent podcast, X said..." |
| **What to cut** | All attribution, podcast context, interview framing |
| **What to amplify** | Your reaction and extension of the insight — add your own experience to it |

**Rule:** Never attribute the insight to the podcast. Make it yours by adding your experience or data.

---

### 6. LinkedIn Comment You Wrote → LinkedIn

| | |
|---|---|
| **Extraction method** | The comment itself is the insight — expand the key point into a full post |
| **Best LinkedIn format** | Deep-Work System Reveal or Hook Stack |
| **Hook angle** | Take the single sentence from your comment that got the most replies — make that line 2 of the hook |
| **What to amplify** | Your reasoning — the comment was short, the post can go deeper |

**Why this works:** Comments that spark replies proved the idea resonates. Expanding them validates the insight before you invest in a full post.

---

### 7. Client Case Study → LinkedIn

| | |
|---|---|
| **Extraction method** | Extract the result + the method. Structure as: problem → what we did → result. |
| **Best LinkedIn format** | Result-First hook + Behind-the-Scenes Story |
| **Hook angle** | Lead with the result (with specifics) — "My client went from X to Y in [timeframe]" |
| **What to cut** | Client name (unless they've given permission and it adds credibility) |
| **What to amplify** | The specific method that produced the result — this is what makes the post unique |

**Privacy rule:** Always get explicit client permission before naming them. If no permission, anonymize with "A [type of] client..." + real results.

---

### 8. Book / Article Takeaway → LinkedIn

| | |
|---|---|
| **Extraction method** | Find the most contrarian or counter-intuitive idea in the book. Ignore the popular highlights. |
| **Best LinkedIn format** | Contrarian Hook + Long-Form Post or Myth-Buster Hook + List |
| **Hook angle** | "I just read [implicit — don't name the book in the hook]. Here's the idea I can't stop thinking about:" |
| **What to cut** | Book summaries — never summarize a book. Extract ONE idea and go deep on it. |
| **What to amplify** | Your personal application of the idea — how does it change how you work or think? |

**Copyright note:** Never quote more than 2-3 words from a book. Reframe ideas in your own words.

---

## Repurpose Quality Checklist

Before publishing any repurposed content:

- [ ] Does the post feel like it came from personal experience? (Not a repurpose?)
- [ ] Is there zero surviving language from the source content?
- [ ] Does it pass the "only I could have written this" test?
- [ ] Have I added something the source didn't have? (Personal angle, specific data, extended insight?)
- [ ] Are there at least 2 viral triggers present?
- [ ] Does the hook work without knowing what the original source was?

If any box is unchecked → rewrite before posting.
